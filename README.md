# Tris (Tic Tac Toe)

Un semplice gioco del Tris (Tic Tac Toe) sviluppato in **Java** con interfaccia grafica (Swing).  
Include **icone e font personalizzati**.

---

## Caratteristiche

- Interfaccia grafica intuitiva con **JFrame, JPanel e JButton**
- Font personalizzato per X e O
- Icona personalizzata della finestra
- Determinazione automatica del vincitore (righe, colonne e diagonali)
- Alternanza dei turni tra X e O
